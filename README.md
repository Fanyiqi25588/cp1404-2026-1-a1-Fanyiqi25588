## Description
Travel Tracker is a Python program that allows users to keep track of places they want to visit.
The program loads place data from a CSV file and allows the user to display places, add new places, get a recommendation for a place to visit, and mark places as visited.

## Features
* Display all places
* Recommend a random unvisited place
* Add a new place
* Mark a place as visited
* Save and load place data from a CSV file

## How to Run
Run the program using:

python assignment1.py

Make sure the file **places.csv** is in the same folder as **assignment1.py**.

## Data Format
The CSV file uses the following format:
Place,Country,Priority,Visited

Example:
Lima,Peru,3,n
Auckland,New Zealand,1,v
Rome,Italy,12,n

Where:
* **n** = not visited
* **v** = visited
